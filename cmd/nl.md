 

-  nl  [OPTION]... FILE...
   
   

- 功能：读取File参数，计算输入中的行号，将计算过的行号写入标准输出。

  

- 参数：

  - -b a，不论是否为空行，也同样列出行号（类似cat -n）
  - -b t，空行不列出行号（默认）
  - -n ln，行号在左
  - -n rn，行号在右
  - -n rn，行号在右且补零
  - -w，补零位数
  
  
  
- 示例:

  - 列出文件内容且带行号：

    ```bash
    nl test.txt
    ```
    
  - 列出文件内容且带行号，空行也带上行号：
    
    ```bash
    nl -b a test.txt
    ```
  - 行号前面补0，统一输出格式：
    
    ```bash
    nl -b a -n rz test.txt
    ```
