 

-  whereis  [options]  [-BMS directory... -f]  name...
   
   

- 功能：定位可执行文件、源码文件、帮助文件在文件系统中的位置。这些文件的属性应该属于原始代码，二进制文件，或是帮助文件。whereis程序具有搜索源代码、指定备用搜索路径和搜索不寻常项的能力。

  

- 参数：

  - -b，定位可执行文件。
  - -m，定位帮助文件。
  - -s，定位源代码文件。
  - -u，搜索默认路径下除可执行文件、源代码文件、帮助文件以外的其他文件。
  - -B，指定搜索可执行文件的路径。
  - -M，指定搜索帮助文件的路径
  - -S，指定搜索源代码的路径
  
  
  
- 示例:

  - 将和ifconfig文件相关的文件都查找出来：

    ```bash
    whereis ifconfig
    ```
    
  - 只讲二进制文件查找出来：
    
    ```bash
    whereis -b ifconfig
    ```
  - 查找说明文档路径：
    
    ```bash
    whereis -m ifconfig
    ```
