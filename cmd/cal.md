- cal [参数] [月] [年]

- 功能：查看日历等时间信息，如果只有一个参数，则表示年份，如果有两个参数，则表示月和年

- 命令参数：

  -  -1，显示一个月的日历
  -  -3，显示系统前一个月，当前月，下个月的日历
  -  -s，显示格式为星期天为第一天
  -  -m，显示格式为星期一是第一天
  -  -j，当年中的第几天
  -  -y，当前年份的日历

- 示例：

  - 显示当前月份日历

    ```bash
    cal
    ```

  -  显示指定月份的日历

     ```bash
     cal 2 2021
     ```

  -  显示某年日历

     ```bash
     cal -y 2021
     cal 2021
     ```

  -  显示自1月1日的天数

     ```bash
     cal -j
     ```
