- wc [OPTION]... [FILE]...

- 功能：统计指定文件中的字节数、字数、行数，如果没有给出文件名，则从标准输入中读取。

- 参数：

  - -c，统计字节数
  - -l，统计行数
  - -m，统计字符数，不能与-c一起使用
  - -w，统计字数。
  - -L，显示最长行的长度

- 示例：

  - 查看文件的行数、单词数、字节数：

    ```bash
    wc test.txt
    ```

  - 只统计数字，不打印文件名：

    ```bash
    cat test.txt | wc
    ```

  - 统计当前目录下的文件数：

    ```bash
    ls -l | wc -l
    ```
