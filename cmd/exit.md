- exit  N

  - 状态码 N 的范围是 0-255，一般情况下，0 表示正常退出，非零表示异常退出。如果是 0-255 之外的数值，则会被强制转换为 uint8_t 类型的数值，比如 -1 会被转换为 255，256 会发生类型宽度截断，被转换为 0；
  - 状态码 N 可以不指定，默认是上一条命令的退出状态码。

- 功能：用于退出当前shell进程

- 示例：

  - 退出终端

    ```bash
    exit
    ```

  - 用于Shell脚本，退出当前Shell进程

    ```shell
    exit 0 #正常退出
    exit 1 #异常退出
    ```
